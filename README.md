# MarketLead.io
Insurance profiling from social media

## Prerequisites
You will need the following things properly installed on your computer.

* [Git](http://git-scm.com/)
* [Node.js](https://nodejs.org/en/) with [npm](https://www.npmjs.com/)
* [Bower](http://bower.io/)

## Installation

* `git clone <repository-url>` this repository
* change to root repo directory
* `npm install`

## Running

* `npm start`

## Testing

### Heroku log tailing
* `heroku loggin`
* `heroku git:remote -a marketlead`
* `heroku logs --tail`

### Automated testing

* [Travis CI](https://travis-ci.com/)

### Testing

* [Mocha](https://mochajs.org/) via [Chai](http://chaijs.com/) with Asserts
* Unit tests: `npm test`
* API tests: `npm run-script apitest`
* Integration tests: `npm run-script integrationtest`
